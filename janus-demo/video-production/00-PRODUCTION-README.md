# Janus Video Production — Master Guide

## Overview
5 promotional videos for Janus v2.0 using Google AI tools (Veo 3.1, MusicFX/Lyria 3, AI Studio, Gemini 3.1 Pro).

## File Index

| File | Contents |
|------|----------|
| `01-dashboard-capture-guide.md` | Step-by-step recording sessions (Day 1) |
| `02-musicfx-prompts.md` | 5 background music prompts (Day 1, parallel) |
| `03-veo-prompts.md` | 8 B-roll clip prompts for Veo 3.1 (Day 2) |
| `04-motion-graphics-specs.md` | AI Studio text cards & animation specs (Day 2) |
| `05-narration-scripts.md` | 5 complete narration scripts with timing |
| `06-edit-decision-lists.md` | Shot-by-shot EDLs with exact timestamps |
| `07-gemini-review-prompt.md` | Final QA prompt for Gemini 3.1 Pro (Day 5) |

| Component | Max Length | Tool |
|-----------|-----------|------|
