# MusicFX / Lyria 3 Prompts — Day 1 (Parallel with Captures)

## Instructions
1. Open [MusicFX](https://aitestkitchen.withgoogle.com/tools/music-fx) or access via Google AI Studio
2. Copy-paste each prompt exactly as written
3. Generate, listen, and regenerate if needed (aim for 2-3 variations per track)
4. Export as WAV or MP3
5. All tracks are 30-second loops — they'll be repeated/trimmed in editing

---

## Track 1 — "The Problem" (Video 2)

**Mood**: Tense, uneasy, building unease
**Tempo**: 80 BPM
**Use in video**: Background for frustrated store owner, empty data, lost revenue stats

### MusicFX Prompt:
```
Subtle tense corporate underscore, minor key, 80 BPM, solo piano with light string pad, ambient, introspective, building tension, no drums, cinematic, 30 seconds
```

### Notes:
- Should feel slightly uncomfortable but not dramatic
- Piano should be sparse — maybe 4-5 notes per bar
- String pad = sustained, not melodic
- Silence between notes is part of the emotion
- Re-generate if it has drums or upbeat energy

### File name: `music-01-problem-tense.wav`

## Quality Checklist Per Track

- [ ] No vocals or humming
- [ ] Loops cleanly (end connects to beginning without jarring transition)
- [ ] No clipping or distortion
- [ ] Volume consistent throughout
- [ ] Mood matches the video context
- [ ] Not too busy — leaves room for voice-over narration
- [ ] Downloaded at highest quality available

## Post-Generation

After generating all 5 tracks:

1. Listen to each back-to-back to ensure they feel cohesive as a set
2. Normalize volumes across all tracks (target -14 LUFS for background music)
3. Test each track with a sample narration to ensure voice clarity
4. Create a backup copy of all selected tracks
